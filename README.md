# TicTacToe

## A console Tic Tac Toe game in c++

### features
* Play against computer
  * Easy and Hard difficulty level
* Undo and Redo
* Back and Quit

### Compile and run
```
g++ main.cpp tictactoe.cpp easylevel.cpp hardlevel.cpp getch.cpp
./a.out
```

### Input for different positions
```
7 8 9
4 5 6
1 2 3
```
