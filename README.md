# TicTacToe

A console Tic Tac Toe game in c++ with feature of play against computer(Easy and Hard level).

Compile and run:
```
g++ main.cpp tictactoe.cpp easylevel.cpp hardlevel.cpp getch.cpp
./a.out
```

**input position:**
```
7 8 9
4 5 6
1 2 3
```
